HTML

HTML (Hypertext Markup Language):

  • HTML is the standard laguage used to create web pages using browser.
  • HTML enhances user experience by making web pages more engaging and interactive.
  • HTML provides specific tags and their attributes for:
    1. structuring contents by defining lists, tables, forms, frames, etc.;
    2. embedding multimedia contents such as image, audio, vedio, animation, marquee, etc.;
    3. formatting texts by making texts bold, italic or underlined or, using different fonts and colors;
    4. creating hyperlinks (using anchor tag) to add links to the web page and navigate links through different websites or resources on internet or on our server;
    5. semantic markup which includes semantic elements like header, nav, main, section, footer, etc. which provide a layout for the html document and help the search engines understand the contents and the contexts of the web page to enhance the search engine ranking of the website {Search Engine Optimization(SEO)}.
